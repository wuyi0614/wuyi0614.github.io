# Yi Wu Academic Website

This repository contains the source code for **Yi Wu's academic profile website**.

The site is being rebuilt as a research-focused homepage centered on:

- sustainable finance
- AI data centers
- city- and corporate-level empirical research
- coding-first academic workflows

## Build and Deployment

The website is built with **Jekyll** and deployed through **GitHub Pages / GitHub Actions**.

The main site content lives in:

- `_pages/`
- `_projects/`
- `_bibliography/`
- `_data/`
- `assets/`

## Acknowledgement

This website is built on top of the open-source [`al-folio`](https://github.com/alshedivat/al-folio) academic Jekyll theme.
